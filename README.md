Este repositorio centraliza las guías, convenciones y recursos esenciales para todos los miembros del equipo.

## Inicio rápido / Quick Start

Si eres nuevo en el equipo, sigue estos pasos en orden:

1. Lee la [Estructura de la organización](docs/organization-structure.md) para entender cómo trabajamos.
2. Configura tu clave SSH siguiendo la [guía de SSH](docs/ssh-setup.md).
3. Revisa los [Primeros pasos con Git](docs/git-basics.md) si no tienes experiencia previa.
4. Ten a mano el [Manual de comandos](docs/git-commands.md) como referencia diaria.
