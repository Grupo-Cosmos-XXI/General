## Inicio rápido para nuevos miembros

1. Configura tu clave SSH siguiendo la [guía de SSH](docs/ssh-setup.md).
2. Revisa los [Primeros pasos con Git](docs/git-basics.md) si no tienes experiencia previa.
3. Lee el [CONTRIBUTING.md](https://github.com/Grupo-Cosmos-XXI/.github/blob/main/CONTRIBUTING.md) para entender cómo trabajamos.
4. Ten a mano el [Manual de comandos](docs/git-commands.md) como referencia diaria.
